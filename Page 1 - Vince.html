Variables

Variables are containers for storing data

In JavaScript you can declare variables in 3 ways
Declaring a variable using 'var' will store data
'var' is more widely compatible with older browsers

If you want to be more specific
Declaring a variable using 'const' stores data and never changes
Declaring a variable using 'let' stores data and can change
